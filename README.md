# ASV Study Bible

Mobile-friendly public-domain ASV study Bible prototype.

Features:

- All ASV verses bundled locally for GitHub Pages use
- Book/chapter/verse dropdown selection
- Reference lookup, e.g. `John 3:16`, `Psalm 23`
- Search across all ASV verses
- Real Hebrew/Greek words loaded by book from MACULA datasets
- Tap/click English or original-language words to highlight linked items where Strong’s-number matching is available

Data sources:

- ASV public-domain text with Strong’s numbers from Bolls Life static translation data
- MACULA Hebrew and MACULA Greek datasets from Clear Bible

Broader English-word-to-original-word alignment is based on Strong’s-number matching, so it is helpful but not always one-to-one.
